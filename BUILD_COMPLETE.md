### Currently Using Mock

- ✅ Context generation → returns deterministic JSON
- ✅ Patent screening → returns consistent results

### To Add Real APIs
1. Get credentials (patent API + AI API)
2. Edit `lib/PatentProvider.php` → `realFetchPatent()`
3. Edit `lib/AiProvider.php` → `realBuildElementContext()` + `realScreenPatent()`
4. Update `config/config.php` with credentials
5. Set `USE_MOCK_* = false`

See **API_INTEGRATION.md** for complete guide.

## 🔧 CONFIGURATION

All settings in one place: `config/config.php`

```php
// Authentication
ADMIN_USERNAME = 'admin'
ADMIN_PASSWORD = 'admin'  
SESSION_TIMEOUT = 3600

// Patent API
USE_MOCK_PATENT_API = true  (change when ready)
PATENT_API_BASE_URL = 'https://...'
PATENT_API_KEY = 'your_key'

// AI API  
USE_MOCK_AI_API = true  (change when ready)
AI_API_BASE_URL = 'https://...'
AI_API_KEY = 'your_key'
AI_MODEL = 'gpt-4'

// Processing
AI_SLEEP_MS = 500  (rate limiting)
BATCH_SIZE = 5
MAX_PATENTS_MVP = 100

// Logging
LOG_API_REQUESTS = true
LOG_PATH = '/path/to/logs/app.log'
```
